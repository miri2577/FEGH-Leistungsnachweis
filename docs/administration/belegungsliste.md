# Belegungsliste pflegen

Die **Teamleitung** pflegt die Stammdaten im Admin-Bereich.

- **Belegungsliste**: Klient*innen anlegen/ändern – AL & kLE (pro Monat), HBG, Bezugsbetreuer, Status.
- **Beendigung**: Status auf *Beendigung* setzen → die Person bleibt sichtbar, zählt aber
  nicht mehr im Teamsitzungs-Teiler.
