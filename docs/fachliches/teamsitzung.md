# Teamsitzung & Gruppenzeit

## Teamsitzung
Jeden **Donnerstag** (3 Std), **Berliner Feiertage ausgenommen**. Die Zeit wird auf die
Klient*innen **in Betreuung** verteilt (Beendete zählen nicht mit):

!!! info "Formel"
    Std je Klient*in = Anzahl Donnerstage × 3 ÷ Klient*innen in Betreuung.

## Gruppen
!!! info "Formel"
    Zeit je Klient*in = Gesamtzeit ÷ Teilnehmer ÷ Anzahl Mitarbeiter.
