# Leistungsnachweis erfassen

1. Öffne **Leistungsnachweis**.
2. Klicke **+ Neue Zeile**.
3. Trage **Datum**, **Klient*in**, **Leistungsart**, **Beginn** und **Ende** ein.
4. Die **Dauer** rechnet das System automatisch; gespeichert wird beim Verlassen der Zelle.

## Leistungsarten
| Kürzel | Bedeutung | zählt als FLS |
|---|---|---|
| FS | fallspezifische Leistung | ja |
| WFS | Vor-/Nachbereitung, Doku | ja |
| BAO | Betreuung am anderen Ort | ja |
| FUS | fallunspezifisch | nein |
| FZ | Fahrtzeit | nein |
| AL | Assistenzleistung | nein |
| KLE | kalkulatorisch (Teamsitzung) | nein |
| FH | Freihaltung/Abwesenheit (50 %) | nein |
